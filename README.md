# ⚖️ 重量小達人 - 兒童數學遊戲

一個幫助兒童學習重量測量的互動式數學遊戲。

## 🎮 遊戲功能

### 題型選擇
- 🎯 **全部混合** - 所有題型隨機出現
- 📊 **讀取磅秤** - 克和公斤磅秤
- ➗ **計算平均** - 除法計算重量
- ⚖️ **天平秤重** - 砝碼加總
- 📏 **比較重量** - 計算重量差異
- 🧩 **複合題** - 磅秤 + 天平

### 難度設定
- 🌟 **簡單模式** - 整數、簡單刻度
- 🔥 **困難模式** - 小數、細刻度

## 🚀 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建構生產版本
npm run build
```

## 📦 部署到 Vercel

### 方法一：GitHub 連接
1. 將此專案推送到 GitHub
2. 登入 [Vercel](https://vercel.com)
3. 點擊 "New Project"
4. 選擇你的 GitHub 倉庫
5. 點擊 "Deploy"

### 方法二：Vercel CLI
```bash
# 安裝 Vercel CLI
npm i -g vercel

# 部署
vercel
```
